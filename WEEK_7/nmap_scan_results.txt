QUESTIONS:
1. What Nmap flag did you use to find the specific software versions running on these ports?
Answer: I used the -sV flag which enables version detection through open port probing, and it identifies the exact service and software version that is currently running on each host.

2. Why is it dangerous to run an intense scan (like -A) on a fragile legacy system?
Answer: Aggressive scans like -A will generate heavy traffic which can potentially lead to more than the older system can handle. The legacy system could be overwhelmed and become unstable in these sorts of environments. It is mostly because these systems don't have the same capabilities as the newer systems which makes them prone to crashing, freezing, or becoming completely unresponsive when it encounters requests that aer usually meant for newer systems.
